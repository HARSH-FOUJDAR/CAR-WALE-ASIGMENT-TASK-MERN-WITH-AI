## Project Overview

This is a full-stack web application that helps users choose the right car based on their preferences such as budget, fuel type, family size, and priority (safety, mileage, performance, features).

The system uses a recommendation engine to filter and rank cars and provides the top matching results. It also uses AI to generate simple explanations for why a specific car is recommended.

---

## Problem Statement

Users often get confused due to a large number of car options, specifications, and price ranges.  
This project solves that problem by helping users get a clear shortlist of suitable cars based on their needs.

---

## Features

- User preference-based car recommendation system
- Top 3 car suggestions based on scoring logic
- AI-generated explanation for recommendations
- Responsive UI for all devices
- Fast and simple decision-making flow
- Clean and minimal user interface

## How It Works

1. User fills a form with preferences (budget, fuel type, family size, priority)
2. Backend reads car dataset
3. Scoring logic ranks all cars
4. Top 3 matching cars are returned
5. AI generates explanation for selected car



This project helps users move from confusion to a clear decision by providing a simple, fast, and AI-assisted car recommendation system.
